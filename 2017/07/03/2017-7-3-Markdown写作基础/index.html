2. 插入图片

2.1 使用关联文件夹

在网上查了很多关于如何在Hexo博客中完美插入本地图片的方法，很多都是说在source下建一个与_posts同级的文件夹，并命名为images，然后在博文中引用其相对路径即可。我反复尝试多次后发现这种方法根本不行（也有可能是我自己的原因）。为了使用户更系统的管理文章资源，Hexo提供了一种文章资源文件夹。文章资源文件夹，顾名思义，就是将一篇博文的所有资源集中在与之同名的一个文件夹中，这里面包括各类图片，一起其它有用处的资源。

使用文章资源文件夹放入本地图片到博客中的方法：

  • 首先需要修改Hexo根目录下的配置文件_config.yml，将post_asset_folder后面的关键字由false改为true（在我装的hexo 3.3.7版本中，post_asset_folder后面已经是true，无需修改）

  • 通过命令：hexo new post article_name 创建新文章时，会同时创建一个与之同名的文件夹，即为资源文件夹
